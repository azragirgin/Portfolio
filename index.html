This is a console-based hangman game implemented in Java, this project contirbuted for my understanding of object-oriented programming principles. The game generates random words for each session and provides a user-friendly interface for guessing within limited attempts or else the user fails to win.

  • Applied object-oriented programming concepts like classes.

  • Practiced creating user-friendly console applications with Java.
