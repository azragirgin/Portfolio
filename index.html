Final Year Project: AI-Based Skin Cancer Detection Tool

This project uses a Convolutional Neural Network (MobileNetV2) to classify skin lesions as benign or malignant. It features a user-friendly Streamlit interface and Grad-CAM visual explanations to improve model transparency and interpretability.

  • Used transfer learning with MobileNetV2 on the ISIC dataset.
  • Implemented Grad-CAM for heatmap visual explanations of predictions.
  • Developed a fully interactive web app using Streamlit.
  • Focused on model training, data augmentation, and class imbalance handling with focal loss.
  • The model has 80% accuracy rate for benign lesions.
